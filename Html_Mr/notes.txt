Web Applications : These are the applications that which are to be called from various places rather than where they are created. 

If you want to make any distributed or robust applications, then we need the help of web applications.

You can create web applications in 2 ways as 

(i) Scripting Languages : These are the languages that which are interpreted, during runtime they takes lot of time to execute. 

(HTML, CSS, JavaScript, Jquery) 

(ii) Programming Languages : These are the languages, that which are compiled first, during runtime they takes very less time to execute. 

(Java, PHP, Python) 

What is Client-Side Scripting and Server-Side Scripting 

Client-Side scripting enables you to access the application from the client-system, No Network traffic and Execution will be faster. 

Server-Side scripting enables you to access the application from the server-system, Network traffic is high, a bit slower.
